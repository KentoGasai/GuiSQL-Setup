# GuiSql

Windows‑приложение для запуска заранее подготовленных SQL‑запросов с ограниченным доступом, хранением подключений и удобной фильтрацией результатов. В комплекте есть утилита для шифрования `.sqlx`.

## Возможности
- Запуск SQL‑запросов из выбранной папки (`.sql` / `.sqlx`)
- Локальная аутентификация пользователя приложения (DPAPI)
- Сохранение логинов/паролей к БД локально (DPAPI)
- Поддержка параметров:
  - обычные `:param`
  - даты `:День::date` (ввод в UI, подстановка `DATE 'YYYY-MM-DD'`)
  - списки `IN :param` (много значений)
- Результаты в таблице + экспорт в Excel
- Фильтрация по колонкам через меню `...` (чекбоксы, поиск, пагинация значений)
- Многоуровневая сортировка с индикаторами
- Сохранение выбранной папки и параметров запросов

## Требования
- Windows 10/11 x64
- .NET SDK установлен **только** для сборки (для запуска опубликованной версии не нужен)
- Доступ к PostgreSQL по сети

## Установщик (Inno Setup)
1) Установить Inno Setup 6  
2) Собрать установщик:
```powershell
cd F:\GitLab\gui-sql\installer
powershell -ExecutionPolicy Bypass -File .\build-installer.ps1 -Version 1.2.0
```
Выход: `installer\output\1.2.0\GuiSqlApp-Setup-1.2.0.exe`
